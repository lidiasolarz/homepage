Czym jest CSS?

Kaskadowe Arkusze Stylów
Odpowiadają za prezentację treści strony, stylowanie - nadawanie wyglądu, wpływa na wygląd elementów HTML na ekranie, wydrukach itp...

Składnia CSS to są reguły:

ZNACZNIK otwierający STYLE
SELEKTOR {
WŁAŚCIWOŚĆ/WŁASNOŚĆ: WARTOŚĆ;
}

Atrybut CLASS

class="name" = .name

Atrybut ID

id="name" = #name

PODSUMOWANIE

Stylujemy po klasach (bez zagnieżdżeń). Nazywmay klasy po angielsku zgodnie z przeznaczeniem. Tworzymy klasy mając nawet jeden alement danego typu.
