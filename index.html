Notatki

Czym jest Frontend?

Odpowiada za wygląd stron i aplikacji internetowych. Odpowiada za interakcję z użytkownikiem. Jest generowany przy użyciu technologii HTML, CSS, JAVASCRIPT. Technologie Frontend-we przejmują rynek tradycyjnych aplikacji.

Warstwy strony internetowej

  • Warstwa 0 - Treść - nośnik informacji, wszystko co widzimy w przeglądarce (CONTENT IS THE KING)
  • Warstwa 1 - Semantyka: HTML - nadaje znaczenie, sens treści poprzez opakowanie jej w znaczniki, które mają swoje znaczenia
  • Warstwa 2 - Prezentacja: CSS - nadaje wygląd niższym warstwom
  • Warstwa 3 - Interakcja: JavaScript - odpowiada za interakcje z użytkownikiem
